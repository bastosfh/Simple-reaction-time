# Simple-reaction-time
The task consists of pressing the switch, as soon as possible, when a rectangle on the center of the screen changes from a bright green color to a bright red color.

After pressing the button, the rectangle changes back to the original green color, and the process can be repeated consecutive times (i.e. the amount of trials can be adjusted by the experimenter), with random time intervals from 1 to 4 seconds between each trial.

Tr_square shows the code for using a LabJack data acquisition (with TTL switch) and Tr_square_mouse for using a mouse to get the response from the participant.
